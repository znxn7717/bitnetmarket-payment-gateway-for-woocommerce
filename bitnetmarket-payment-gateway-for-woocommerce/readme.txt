== Short Description ==
درگاه پرداختی که کسب‌وکار شما را از بانک‌ها رها می‌کند
== Description ==
درگاه پرداخت بیت‌نت‌مارکت امکان پذیرش رمزارزها را به سادگی و امنیت کامل به فروشگاه شما اضافه می‌کند. روش پرداخت با رمزارزهای مختلف را در سایت خود فعال کرده و تجربه خریدی مدرن، سریع و بدون مرز برای مشتریان‌تان فراهم کنید. راه‌اندازی آنی، تراکنش‌های جهانی، آزادی مالی.

== Installation ==
1. پوشه `bitnetmarket-payment-gateway-for-woocommerce` را در `/wp-content/plugins/` آپلود کنید.
2. افزونه را فعال کنید.
3. در WooCommerce > Settings > Payments > Bitnetmarket شناسه فروشنده را وارد کنید.
