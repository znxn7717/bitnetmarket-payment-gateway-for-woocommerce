== Description ==
درگاه پرداخت بیت‌نت‌مارکت امکان پذیرش پرداخت‌های رمزارزی را به سادگی و امنیت کامل به فروشگاه اینترنتی شما اضافه می‌کند. با استفاده از این افزونه می‌توانید رمزارزهای مختلف را به عنوان روش پرداخت در سایت خود فعال کرده و تجربه خریدی مدرن، سریع و بدون مرز برای مشتریان‌تان فراهم کنید

== Installation ==
1. پوشه `bitnetmarket-payment-gateway-woocommerce` را در `/wp-content/plugins/` آپلود کنید.
2. افزونه را فعال کنید.
3. در WooCommerce > Settings > Payments > Bitnetmarket شناسه فروشنده را وارد کنید.
